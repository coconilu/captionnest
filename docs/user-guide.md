# 用户指南（Windows M1–M5）

## 从安装到字幕

```mermaid
flowchart LR
    A["安装当前用户 NSIS"] --> B["启动并查看环境检测"]
    B --> C["下载识别模型"]
    C --> D["调整新任务默认配置"]
    D --> E["选择视频并开始处理"]
    E --> F["按流水线查看每步状态与产物"]
    F --> G["得到一个双语 SRT"]
```

## 默认配置与任务配置

| 配置层 | 何时使用 | 是否记住 |
|---|---|---|
| 新任务默认配置 | 创建下一个任务时复制 | 会；模型、语言、Provider、Endpoint、超时和导出偏好会在重启后恢复 |
| 当前任务步骤配置 | 只影响当前任务，可在媒体、识别、翻译、导出卡片中修改 | 会；与任务一起保存，不影响新任务默认值 |
| DeepSeek API Key | 当前页面中下一次执行 DeepSeek 时使用 | 不会；刷新或退出后必须重新填写 |

新建任务仍会自动从第一步连续执行。只有在失败后修复配置，或主动修改某一步时，才需要使用步骤卡片上的“从此步骤重试/继续”。

## 失败后继续

| 失败步骤 | 可以修改 | 重试时复用 |
|---|---|---|
| 媒体准备 | 更换视频 | 无；从新媒体重新开始 |
| 语音识别 | 模型、CPU/CUDA、字幕切分、动态切片边界、VAD、Beam Size | 已完成的媒体检查 |
| 字幕翻译 | 目标语言、Provider、模型、Endpoint、超时；DeepSeek Key 单独填写 | 已完成的识别文本和时间轴 |
| 字幕导出 | 输出目录、是否覆盖同名文件 | 已完成的识别与翻译产物 |

步骤卡片会显示“配置 vN”“执行 N 次”、当前错误和产物摘要。修改上游配置后，受影响的下游步骤会显示为等待或已过期；此前成功的无关上游步骤保持完成状态。

| 识别开关 | 默认 | 作用 | 关闭后的行为 |
|---|---|---|---|
| 动态切片边界 | 开 | 将 60 秒目标边界吸附到附近自然停顿，核心保持 45–75 秒 | 使用原有固定 60 秒切片 |
| 语音活动检测 | 开 | 交给 Faster-Whisper 跳过长静音区域 | 保留静音参与模型识别；不影响动态边界开关 |

动态边界找不到合适停顿、VAD 调用失败或音频短于 45 秒时，会自动使用固定切片，不需要手动重试。
升级前创建且没有该开关记录的旧任务继续按固定切片显示和重跑；只有新建任务默认启用动态边界。

## 环境卡片怎么处理

| 状态 | 含义 | 操作 |
|---|---|---|
| 本地运行时可用 | 安装包 sidecar 正常 | 无需操作 |
| 模型缺失 | 还没有所选识别模型 | 点击“下载模型”，等待完成 |
| CPU | 可正常使用，但长视频速度较慢 | 直接使用；CUDA 是可选优化 |
| Codex 未安装 | 只有 Codex Spark 不可用 | 点击官方安装入口；安装后点“重新检测” |
| Codex 未登录 | CLI 已安装但无 ChatGPT 登录 | 在终端执行 `codex login`，然后重新检测 |
| 媒体运行时异常 | PyAV/FFmpeg 打包不完整 | 重新安装官方 Release 并附版本报告问题 |

应用不会自动安装 Codex，也不会读取或保存 Codex 密码。LM Studio、DeepSeek 可在 Codex 不可用时独立使用。DeepSeek API Key 只随执行请求进入内存，不写入默认配置、任务记录或日志。

## 输出规则

| 项目 | 规则 |
|---|---|
| 文件名 | `<视频名>.srt`，例如 `movie.srt` |
| 目标语言 | 不写入文件名；以任务中选择的语言为准 |
| 逐词重排 | 默认；按逐词时间戳切开长静音，适合直接观看 |
| 分片原始段 | 保留模型原始段落边界，适合诊断对照 |

每个任务只写一个文件。字幕块先写源语言，再换行写目标语言。如果同目录已有同名字幕，生成结果会覆盖该文件。如果源语言自动检测结果与目标语言相同，应用会提示更换目标语言，不会生成文件。

使用“选择本地视频”时，字幕默认写入视频同目录。普通拖放上传无法取得浏览器外的原绝对路径，因此应用会复制视频到应用数据目录，并在副本旁输出字幕。

## 模型与磁盘

| 模型 | 取舍 |
|---|---|
| small | 默认选项；下载小、CPU 更快、准确率相对低 |
| medium | 速度与准确率折中 |
| large-v3-turbo | 较快的大模型选择 |
| large-v3 | 高质量，下载和计算成本最高 |

下载前应用会检查磁盘空间。下载中断不会覆盖已可用模型；缺失或损坏时可重新下载。模型许可证由模型发布者决定，下载前应查看对应模型卡。

## 常见问题

| 现象 | 建议 |
|---|---|
| 安装时出现 SmartScreen | M5 尚未配置 Authenticode；只从可信 Release 下载并核对 `.sha256` |
| 第一次启动较慢 | sidecar 与 WebView2 正在初始化；超过 30 秒仍无窗口请重新安装 |
| CPU 处理很慢 | 先改用 small/medium；GPU 支持是可选增强，不影响基本使用 |
| 字幕跨越很长静音 | 选择“逐词重排”；“分片原始段”主要用于诊断 |
| 目录不可写 | 将视频移到当前用户有写权限的目录，或使用拖放副本模式 |
| 翻译失败 | 检查 Codex 登录、LM Studio 服务或 DeepSeek 网络/API Key；修改后在“字幕翻译”步骤重试，不会重跑识别 |
| 导出失败或目录不可写 | 修改“字幕导出”的输出目录后从该步骤重试，不会重跑识别和翻译 |
| 历史任务提示 Qwen3-ASR 已停用 | 任务记录和已有产物仍会保留；在“语音识别”步骤改选任一 Faster-Whisper 模型并保存后即可继续 |
| Codex 安装后仍显示不可用 | 完全退出应用后重新打开，或先点“重新检测”；确认终端可执行 `codex --version` |
