# CaptionNest Web UI 设计规格

## 目标

首版是一个本机运行的单屏工具界面：选择视频，选择目标语言与翻译方式，查看处理进度，并在完成后打开唯一双语字幕所在目录。

## 视觉基准

- 基准图：`docs/design/concept.png`
- 画布：桌面端 1440px 级别单屏，响应式收窄到移动布局。
- 背景与表面：白色；正文：深炭黑；辅助文字：石板灰。
- 主色：CaptionNest 青 `#0B8492`；亮青只用于品牌图形，绿色仅表示成功。
- 圆角：控件 10px，主要面板 12px。
- 容器：开放式左右两列，左侧工作区、右侧设置栏；不增加侧边导航和营销区块。

## 必需界面

1. 顶栏：产品名、本地环境/CUDA/后端状态。
2. 视频区域：系统文件选择、选中文件信息。
3. 任务流水线：媒体准备、语音识别、字幕翻译、字幕导出四张步骤卡；每张卡展示状态、配置版本、执行次数、产物和错误，并支持从该步继续。
4. 处理日志与总体进度。
5. 默认设置栏：目标语言（简体中文/英语/韩语）、Whisper 模型、CPU/CUDA 状态；翻译 Provider、模型、Endpoint、API Key；非密钥选项会被记住并复制给新任务。
6. 完成态：唯一双语 SRT 路径、打开文件夹操作。
7. 环境区：sidecar、识别模型、媒体运行时和 Codex 状态；支持模型下载与“重新检测”。

默认设置和任务设置必须在视觉上明确分层。任务创建后，各步骤配置与任务绑定；修改任务配置不反向覆盖默认设置。DeepSeek API Key 只保留在当前页面内存中，不能进入默认设置持久化、任务详情或步骤执行记录。

## Provider 显示规则

| Provider | 模型 | Endpoint | API Key |
|---|---|---|---|
| Codex Spark | `gpt-5.3-codex-spark` | 隐藏 | 隐藏 |
| LM Studio | 用户填写本地模型 ID | 默认 `http://127.0.0.1:1234/v1` | 可空 |
| DeepSeek | 默认 `deepseek-v4-flash` | 默认 `https://api.deepseek.com` | 必填 |

## 浏览器边界

源语言始终自动检测，不显示识别语言选项；输出始终为原文在上、目标语言在下的单个 SRT，不显示“写出原文字幕”等开关。

本机文件选择器由后端弹出系统对话框，直接使用原视频并把字幕写回视频同目录。
