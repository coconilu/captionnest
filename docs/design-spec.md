# SubLingo Local Web UI 设计规格

## 目标

首版是一个本机运行的单屏工具界面：选择视频，配置识别与翻译，查看三阶段进度，并在完成后打开字幕所在目录。

## 视觉基准

- 基准图：`docs/design/concept.png`
- 画布：桌面端 1440px 级别单屏，响应式收窄到移动布局。
- 背景：`#f7f8fa`；表面：白色；正文：深炭黑；辅助文字：石板灰。
- 主色：靛蓝紫；绿色仅表示成功。
- 圆角：控件 10px，主要面板 12px。
- 容器：开放式左右两列，左侧工作区、右侧设置栏；不增加侧边导航和营销区块。

## 必需界面

1. 顶栏：产品名、本地环境/CUDA/后端状态。
2. 视频区域：拖放区、系统文件选择、选中文件信息。
3. 流水线：提取音频、语音识别、翻译字幕。
4. 处理日志与总体进度。
5. 设置栏：识别语言、Whisper 模型、CUDA；翻译 Provider、模型、Endpoint、API Key；输出选项。
6. 完成态：原文字幕与中文字幕路径、打开文件夹操作。

## Provider 显示规则

| Provider | 模型 | Endpoint | API Key |
|---|---|---|---|
| Codex Spark | `gpt-5.3-codex-spark` | 隐藏 | 隐藏 |
| LM Studio | 用户填写本地模型 ID | 默认 `http://127.0.0.1:1234/v1` | 可空 |
| DeepSeek | 默认 `deepseek-v4-flash` | 默认 `https://api.deepseek.com` | 必填 |

## 浏览器边界

- 本机文件选择器：由后端弹出系统对话框，直接使用原视频并把字幕写回视频同目录。
